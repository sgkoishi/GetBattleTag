# Get BattleTag

 [![GitHub release](https://img.shields.io/github/release/sgkoishi/GetBattleTag.svg?maxAge=604800)](https://github.com/sgkoishi/GetBattleTag/releases)

Get BattleTag is a plugin for [Hearthstone Deck Tracker](https://github.com/HearthSim/Hearthstone-Deck-Tracker) that allows you to get your current opponent BattleTag.

After installing the plugin, go to 'PLUGINS > Get BattleTag' to copy your opponent's BattleTag to your clipboard.

This is a reimplement Get BattleTag of [Kim Tranjan's version](https://github.com/KimTranjan/GetBattleTag) based on HDT's API.
